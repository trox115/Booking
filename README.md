# Barber Booking App
An app that allow users to book a barber. The purpose of the entire project is to work with a full-stack ( ruby on rails and react).


On a new version it will allow the barber to register too.

## Built With
- Javascript
- React
- React-redux
- React-router
- npm
- Ruby on Rails
- CSS 
- Html

## Getting Started
